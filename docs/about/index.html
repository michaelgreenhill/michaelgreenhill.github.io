About

My name is Michael and I'm passionate about IT infrastructure.

I built this site using 11ty.dev and it's hosted on Amazon S3, served to you with an attractive domain name via CloudFront. The site is developed locally on a Windows 10 desktop in Selby, Victoria, and commited to Github where the build process ships it to S3.

You can read more about my professional adventures over on LinkedIn, or check out my Github misadventures.

I'm a practicing AWS Solutions Architect for a leading consumer finance company in Melbourne's outer east. When I'm not mashing my head into a keyboard I'm usually outside taking photographs of steam trains.

I've done loads of cool things in my personal and professional lives. One of the achievements I'm most proud of was being asked to present at the 2018 Railway Photography Forum.
